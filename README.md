#CloudStore

## **Tech Stack**

- React
- React Context API
- React router
- Razorpay for Payments
- CloudUI

## **Key Features**

- Filter Products By

  - Price Trend (high to low/low to high)
  - Ratings
  - Price Range
  - Categories

- Auth

  - Login
  - Logout
  - Signup

- Cart

  - Add to Cart
  - Remove from Cart
  - Change quantity

- Wishlist

  - Add to Wishlist
  - Remove from Wishlist
  - Add to Cart

- Address
  - Add Address
  - Delete Address
  - Update Address
- Payments
- User Profile
